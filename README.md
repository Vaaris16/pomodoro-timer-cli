# Pomodoro-timer-cli[RUST]

# About

A simple Pomodoro Timer right in your terminal built in rust. Uses figlet-rs to render an ASCII timer in your terminal and plays an alarm when state changes.

![App Demo](./assets/examples/work_example.png)
![](./assets/examples/rest_example.png)

# Installation

```bash
cargo install pomodoro-timer-cli
```

# Flags
- `-a`, `--alarm` — Enables alarm which plays an alarm on state change

Enjoy!
